# ✨ Glowing Buttons

A collection of modern, stylish **glowing buttons** built using HTML, CSS, and JavaScript.
These buttons feature smooth animations, neon effects, and customizable designs to enhance your UI.

---

## 🚀 Features

* 🔥 Neon glow effects
* 🎨 Fully customizable styles
* ⚡ Smooth hover animations
* 💻 Pure HTML, CSS & JavaScript (no frameworks)
* 📱 Responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations, Gradients, Shadows)
* JavaScript (optional effects)

---

## 📂 Project Structure

```
glowing-buttons/
│── index.html
│── style.css

│── preview.png
```

---

## ⚙️ How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/madhvithalapadiya/glowing-buttons.git
   ```

2. Open the project folder:

   ```bash
   cd glowing-buttons
   ```

3. Run the project:

   * Open `index.html` in your browser
